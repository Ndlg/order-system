# 阶段 29：商品识别页读取采集 Excel

## 背景

客户在配置“抖店商品标题关键词 -> 我的商品”时，不应该完全靠手动输入。

更顺的流程是：

```text
上传采集导出的 Excel
系统读取已解析的商品标题列
客户从真实面单商品标题里选择一条
回填为商品识别关键词
客户再选择自己的商品并保存规则
```

## 目标

```text
商品识别页支持上传 .xlsx。
后端读取 Excel 第一张表。
自动识别商品标题列、规格列、数量列。
前端展示商品标题候选。
点击“使用”回填到商品识别关键词输入框。
不自动批量生成规则。
```

## 已完成

```text
新增 POST /api/v1/matching/excel-preview。
后端使用 openpyxl 读取 xlsx。
支持识别商品标题 / 商品信息 / 商品名称 / 商品简称 / 商品显示文字等列名。
支持读取规格列和数量列。
重复的商品标题 + 规格会合并并统计出现次数。
商品识别页新增“读取采集 Excel”上传区域。
上传后展示文件名、表名、识别到的商品列和规格列。
候选表支持点击“使用”回填商品关键词。
```

## 当前口径

```text
Excel 读取只是辅助录入。
用户仍然需要选择关联商品并点击保存规则。
系统不会根据 Excel 自动创建商品识别规则。
第一版只支持 .xlsx。
```
