# 阶段 39：客户管理端打印模板规则 UI

> 日期：2026-06-12  
> 状态：已完成本轮最小版本

## 背景

三种面单模式都需要进入同一个客户理解路径：

```text
面单模式 + 模板识别码 + 工作区 = 当前客户自己的模板配置对象
```

其中抖店和菜鸟店铺直打已经由平台解析出结构化字段；菜鸟 woda / 我打中转只能读取自定义打印区文字，需要客户按自己的模板定义拆分规则。

## 本轮目标

```text
新增客户管理端“打印模板规则”页面。
按当前工作区已采集的 standard_details 汇总模板组。
展示三种模式的模板识别状态。
结构化模板只显示平台字段样例。
我打模板展示自定义区样本，并提供行级拆分规则配置和预览。
配置保存到当前工作区，不能跨客户共享。
```

```text
页面按模板组展示样本数量和状态。

我打中转显示自定义区行文字、商品文字行、规格文字行、数量和尺码预览。
旧解析样本不会把菜鸟标准模板 URL 当作我打模板识别码展示。
```
